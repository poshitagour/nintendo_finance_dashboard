AI Reflection & Methodology

AI Reflection: Use of ChatGPT in This Project

Overview of AI Use

I treated ChatGPT as an assistant across the project, but I kept control of the decisions. Every output required review, testing, and adjustment before use.

How I Used AI in My Workflow

I used ChatGPT to support practical tasks, including:

  • Interpreting the assignment brief and rubric in clear steps
  • Creating starter code for HTML, CSS, JavaScript, and Plotly charts
  • Debugging issues when charts failed to render or broke layout
  • Improving insight wording to sound business-focused and concise
  • Structuring the dashboard flow and section hierarchy

I reviewed all outputs line by line. I modified logic, data references, and wording to align with real data and the project goal.

What I Learned About Prompting

Prompt quality shaped output quality. Short prompts produced generic responses. Detailed prompts produced targeted results. Effective prompts included:

  • Clear business context
  • The exact task or problem
  • Constraints from the assignment brief
  • Expected output format

This approach reduced rework and improved accuracy.

The Challenges I Encountered and How They Were Addressed

Challenges arose when AI suggested using localStorage for the simulator, which doesn't work in single-file HTML contexts. I had to debug this myself by reading MDN documentation and realizing I needed to use in-memory variables instead. Another challenge was AI generating overly complex Chart.js annotations; I simplified these by asking for "minimal code that just draws a horizontal line."

I solved these through source verification, manual testing, and step-by-step debugging with AI guidance.

What I Learned Myself vs AI Support

I handled:

  • Data interpretation
  • Business reasoning
  • Insight validation
  • Alignment with grading criteria

AI supported:

  • Code structure
  • Debugging logic
  • Chart configuration examples
  • Text clarity improvements

This project built confidence in using AI as a productivity tool while retaining full ownership of analysis and decisions.
